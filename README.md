# Excellens Colaborador

Aplicativo de controle de ponto eletrônico desenvolvido para uso corporativo,
com funcionamento offline e interface moderna.

## Funcionalidades
- Registro de entrada, almoço, retorno e saída
- Cadastro de colaboradores com foto
- Painel administrativo com senha
- Geração de folha de ponto em PDF
- Funciona offline após o primeiro acesso

## Tecnologias
HTML, JavaScript, React via CDN, Tailwind CSS e jsPDF.

Projeto voltado para uso interno empresarial.
